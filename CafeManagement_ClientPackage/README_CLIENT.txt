CAFÉ MANAGEMENT SYSTEM - QUICK START GUIDE

=== INSTALLATION ===
1. Double-click 'cafe-management.exe' 
2. Windows may show security warning - click 'More info' then 'Run anyway'
3. Application starts automatically (takes 30-60 seconds first time)

=== FIRST LOGIN ===
1. Open any web browser (Chrome, Firefox, Edge)
2. Go to: http://localhost:5000
3. Login credentials:
   Username: admin
   Password: admin123
4. IMPORTANT: Change password immediately after first login!

=== DAILY OPERATIONS ===
- Start: Double-click 'cafe-management.exe' when café opens
- Access: Go to http://localhost:5000 in browser
- Multiple staff can use different browsers/tabs
- Close: Right-click system tray icon and select 'Quit'

=== YOUR DATA ===

- Backup this file daily to safe location
- No internet required - everything works offline
