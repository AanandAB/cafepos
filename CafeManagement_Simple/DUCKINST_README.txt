DUCKINST - Automated Cafe Management Installer

WHAT IS DUCKINST?
Automated installer that sets up your entire cafe management system with one click.

HOW TO USE:
1. Install Node.js from https://nodejs.org first
2. Choose your installer:
   - Windows: Double-click DUCKINST.exe or DUCKINST.bat
   - Any system: Run python DUCKINST.py
3. Follow the on-screen instructions
4. Duck will install everything automatically

WHAT DUCKINST DOES:
- Checks if Node.js is installed
- Installs all required dependencies
- Builds the complete application
- Creates database file
- Makes START_CAFE.bat launcher
- Sets up everything ready to use

AFTER INSTALLATION:
- Double-click START_CAFE.bat to run your cafe system
- Open browser to http://localhost:5000
- Login with admin / admin123

The duck makes cafe setup quack-simple!
