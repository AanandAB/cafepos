CAFE MANAGEMENT SYSTEM - SIMPLE VERSION

REQUIREMENTS:
- Windows computer
- Node.js (download from https://nodejs.org)

SETUP:
1. Install Node.js version 18 or higher
2. Double-click start-cafe.bat
3. Wait for "serving on port 5000" message
4. Open browser to http://localhost:5000
5. Login: admin / admin123

DAILY USE:
- Morning: Double-click start-cafe.bat
- Evening: Close the black command window

This is the simplest way to run your cafe system without complex executables.
